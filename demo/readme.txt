QR Code Vethica

Démontration
Le répertoire demo contient des QR Codes d'essais.

Certains QR Code pointent sur des enregistrements existants. Scanner ces QR Code affichera la fiche de l'animal

D'autres QR Code sont "vierges" i.e. ils n'existent pas dans le base. Scanner ces QR Code déclenchera le processus 
d'identification du propriètaire et d'enregistrement des informations sur l'animal.
